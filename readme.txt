== Description ==

This plugin as a addons for the plugin <a href="https://wordpress.org/plugins/wp-postviews/" target="_blank">WP-PostViews</a> Must be you need to installed and properly configure the (<a href="https://wordpress.org/plugins/wp-postviews/" target="_blank">WP-PostViews</a>) plugin. If you've configured properly then our addons plugin will be work.
